## Possible formatting codes

| Code | Type          | Returned Value | Preview                 |
|------|---------------|----------------|-------------------------|
| `k`  | Obfuscated    | `§k`           | `(No preview possible)` |
| `l`  | Bold          | `§l`           | **Bold**                |
| `m`  | Strikethrough | `§m`           | ~~Strikethrough~~       |
| `n`  | Underlined    | `§n`           | <u>Underlined</u>       |
| `o`  | Italic        | `§o`           | *Italic*                |
